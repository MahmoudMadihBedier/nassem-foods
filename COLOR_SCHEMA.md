# Naseem Al-Zabadani Color Schema

## Extracted from Logo

The color palette has been carefully extracted from the Naseem Al-Zabadani logo and applied throughout the project for a cohesive, vibrant brand experience.

### Primary Colors

**Primary (Golden Orange)**: `#F4A623`
- Main brand color inspired by the warm spice tones
- Used for: Buttons, links, highlights, accents
- Variations:
  - 50: `#FFF7ED` (lightest)
  - 100: `#FFEDD5`
  - 200: `#FED7AA`
  - 300: `#FDBA74`
  - 400: `#FB923C`
  - DEFAULT: `#F4A623`
  - 600: `#EA580C`
  - 700: `#C2410C`
  - container: `#D97706`

**Secondary (Coral Red)**: `#E85C41`
- Secondary accent derived from the logo's red tag and Arabic text
- Used for: Secondary buttons, highlights, alternative CTAs
- Variations:
  - 50: `#FEF2F2` (lightest)
  - 100: `#FEE2E2`
  - 200: `#FECACA`
  - 300: `#FCA5A5`
  - DEFAULT: `#E85C41`
  - 500: `#EF4444`
  - 600: `#DC2626`
  - 700: `#B91C1C`
  - container: `#991B1B`

**Accent (Vibrant Orange)**: `#FF6B35`
- Energetic accent color for special elements
- Used for: Badges, callouts, decorative elements
- Dark variant: `#D84315`

### Neutral Colors

- **Background**: `#FFFFFF`
- **Surface**: `#F9F9F9`
- **Surface Container** (various depths):
  - lowest: `#FFFFFF`
  - low: `#F5F5F5`
  - DEFAULT: `#EEEEEE`
  - high: `#E0E0E0`
  - highest: `#D6D6D6`

## Design Philosophy

The color scheme creates a warm, inviting atmosphere that reflects the Middle Eastern culinary heritage while maintaining modern luxury. The gradient applications between primary (golden) and secondary (coral) colors create visual interest and guide user attention to important elements.

The vibrant warm tones evoke spices, tradition, and the weightless dining experience theme of Naseem Al-Zabadani.
